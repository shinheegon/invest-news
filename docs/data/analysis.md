🔴 **비관론자**: 문제는 **가격이 아니라 자금 흐름**이다. 외국인+기관 합쳐 8.7조 순매도, 환율 1,540원, 외인 삼성전자 보유율 13년 최저. 매파 워시 연준은 **'워시 풋'이 없다**고 못박았고, 7월 국민연금 리밸런싱 매물이 대기 중. 단일종목 레버리지 25% 급락은 **반대매매 도미노**의 신호일 수 있다. 3. **오늘 한국장 방향성: 중립~하방(변동성 극대)**. 추격매도·추격매수 모두 위험.
   - **조건부 시나리오**: ① 내일 외국인 순매도 둔화 + 환율 1,540 아래 안착 → 기술적 되돌림. ② 환율 1,540 돌파 + 美 PCE 추가 매파 → 변동성 2차 확대.
