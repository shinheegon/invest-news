## 2. 🧠 선행신호 패턴 라이브러리
1. **첫 대형 고객사 수주/공급계약**(실적 반영 전) → 과거: 수주공시 후 분기 실적 확인되며 재평가. *예: 제이앤티씨 2,500억 커버글라스 수주.*
2. **캐파(증설)·자금조달(BW/유증)** → 증설은 수요 선반영. *예: 네온테크 150억 BW.*
3. **성장 테마 밸류체인 첫 거론**(언급 0→1) → 테마 편입 초기가 가장 쌈. *예: 코닝 韓 유리기판 투자 검토 → 유리기판 테마.*
4. **정부 인증·국책과제 주관 선정** → 레퍼런스·매출 가시성. *예: 한울반도체 AI팩토리, 이엠앤아이 제조로봇.*
5. **글로벌 고객 검증(데모/PoC) 진입** → 납품 직전 단계. *예: 마이크로디지탈 SII 최종 검증.*
6. **자회사·지분가치 재평가 리포트**(시총 < 부분가치) → 가치 괴리 해소 기대. *예: 위지트.*
7. **수급 전환 초기**(기관·외국인 순매수 막 시작) — 급락장에선 신뢰도 낮아짐(전반적 매도).
